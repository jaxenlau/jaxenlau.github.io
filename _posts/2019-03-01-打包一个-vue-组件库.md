打包一个 vue 组件库 ([@vue/cli-server](https://cli.vuejs.org/zh/guide/build-targets.html#%E5%BA%93))

安装 vue-cli-server
``` shell
 yarn global add @vue/cli-service

```

构建 lib
```
vue-cli-service build --target lib --name myLib xxxComponent.vue
```
